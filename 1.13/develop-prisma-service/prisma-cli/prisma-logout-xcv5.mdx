## Description

Logout from [Prisma Cloud](https://www.prisma.io/cloud). 

This command simply deletes the `cloudSessionKey` from `~/.prisma/config.yml`.

## Usage

```sh
prisma logout
```

## Examples

**Log out of Prisma Cloud**

```sh
prisma logout
```
